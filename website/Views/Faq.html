FAQ

How does the extension understand my project's specific architecture?

IntelliTest scans your project structure, dependencies, and code patterns to build the context it needs before generating tests. That wider view helps it match your repository's conventions and produce output that fits the code you already have.

What types of tests can the LLM generate?

You can generate unit, integration, regression, and end-to-end tests. IntelliTest uses the code context and your prompt to shape the output around the kind of coverage you need.

Can I export or share the generated test cases?

Yes. You can export generated test cases and scripts so they are easier to review, share, and track outside the editor.

How does the "Agent" approach differ from standard Copilot chat?

Copilot chat is interactive and conversational. IntelliTest is built to follow a structured workflow that gathers context, shapes the request, and returns test code in a more guided way.

Does the extension work with any programming language or framework?

Yes. IntelliTest adapts to the language and testing framework it detects in your project, so the generated output matches the syntax and structure your codebase already uses.
